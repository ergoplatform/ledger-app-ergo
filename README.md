# Ergo Ledger Application

This is a work-in-progress Ergo application for Ledger Nano S.

# Development notes

- [Ledger documentation](https://ledger.readthedocs.io/)
- [Development environment setup](https://ledger.readthedocs.io/en/latest/userspace/getting_started.html)

# Schnorr Signatures

This application successfully demonstrates computing a Schnorr signature for a
32-byte message on a Ledger device, currently demonstrated via
PRINTF-debugging.
